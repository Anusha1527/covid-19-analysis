# COVID-19 Impact Analysis and Prediction

## Description
This project analyzes the impact of COVID-19 by visualizing infection and recovery trends using real-world data. It also predicts the number of COVID-19 cases expected in the next 7 days using time series forecasting.

## Dataset
- covid_19_clean_complete.csv

## Tools Used
- Python
- Pandas
- Plotly
- Facebook Prophet

## Features
- Infection trend visualization
- Recovery trend visualization
- 7-day future case prediction

## Platform
Developed using Google Colab
